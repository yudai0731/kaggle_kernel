## EDA
どんな特徴量があるか?(Target :1 Feature : 6)
### 数値データ
- price : Target,商品の最終入札額
- shipping cost : 送料

### カテゴリカルデータ
- shipping cost : 1:送出品者負担 0:送料落札者負担
- item_condition_id : 商品の状態
- name : 商品の名前
- brand_name : ブランド名
- category_name : カテゴリの名前 single/multiple
- item_description : 商品の説明
